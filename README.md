# User Data Fetcher

A simple JavaScript application that fetches user data from JSONPlaceholder API and displays it on a web page.

## Features
- Fetches user data using async/await
- Displays user name, email, and city
- Error handling with try/catch
- Loading indicator while fetching
- Responsive design

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6+)

## How to Run
1. Clone the repository
2. Open index.html in a web browser
3. Click the "Fetch Users" button or wait for auto-fetch on page load

## API Reference
Uses JSONPlaceholder API: https://jsonplaceholder.typicode.com/users
